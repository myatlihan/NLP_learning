# NLP Learning Repository

This repository contains my implementations and experiments while learning Natural Language Processing.

## Topics

- Text Preprocessing
- Bag of Words
- TF-IDF
- Word Embeddings (Word2Vec, FastText)
- N-Grams
- Hidden Markov Models
- POS Tagging
- Dependency Parsing
- Named Entity Recognition
- Information Retrieval

## Installation

```bash
pip install -r requirements.txt
```

## Run Example

```bash
python text_representation/tfidf_example.py
```
